## 🚀 Déploiement rapide

### Option 1 : GitHub Pages (Gratuit)
1. Fork ce repo
2. Activer GitHub Pages dans Settings &gt; Pages &gt; Source: main branch
3. Votre site est live !

### Option 2 : Vercel (Recommandé pour SEO)
1. Cliquez sur le bouton "Deploy with Vercel" ci-dessus
2. Connectez votre compte GitHub
3. Déployé en 30 secondes !

## 🛠️ Stack technique

- **Frontend** : HTML5 + Tailwind CSS (CDN)
- **Logique** : JavaScript Vanilla (ES6+)
- **Hébergement** : GitHub Pages / Vercel
- **Analytics** : Google Analytics (à ajouter)
- **Monétisation** : Google AdSense + Liens d'affiliation
